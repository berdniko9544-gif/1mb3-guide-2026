# 1MB3 Landing 2026 (Next.js 15 + React 19 + Tailwind v4) + AI-консультант

## Запуск
```bash
npm i
npm run dev
```

## Где поменять важное
- Оплата: `lib/config.ts` (paymentUrl)
- Контакты: `lib/config.ts`
- Демо-видео: `lib/config.ts` (demoVideoUrl)
- Контент/FAQ/знания для чата: `lib/knowledge.ts`

## AI-чат (бесплатный)
По умолчанию чат работает **без внешних API** (rule-based + мини-поиск по базе знаний).
Это стабильно и бесплатно.

Если захотите подключить настоящий LLM:
- добавьте переменную окружения `OPENAI_API_KEY`
- и задайте `OPENAI_BASE_URL` (опционально, если используете прокси/совместимый API)

> Код уже предусматривает это, но по умолчанию будет работать без ключей.
