## Как развернуть на хостинге

### Вариант 1: Простой хостинг (без Node.js)
Если ваш хостинг не поддерживает Node.js, используйте **статический экспорт**:
1. Локально выполните: `npm run build && npx next export`
2. Загрузите содержимое папки `out/` на хостинг
3. Настройте веб-сервер (Apache/Nginx) на эту папку

### Вариант 2: Хостинг с Node.js
1. Распакуйте `1mb3-production-ready.zip` на сервере
2. Запустите: `node .next/standalone/server.js` (если есть standalone build)
   ИЛИ: `npm start` (использует уже собранный .next)
3. Сайт будет доступен на порту 3000

### Вариант 3: Vercel/Netlify (рекомендуется)
1. Загрузите исходники (без node_modules и .next)
2. Платформа автоматически соберет проект
3. Деплой займет 1-2 минуты

## Переменные окружения
Если нужны переменные окружения, создайте файл `.env.local`:
```
NEXT_PUBLIC_API_URL=your_api_url
```

## Порт и запуск
По умолчанию Next.js использует порт 3000.
Для изменения порта: `PORT=8080 npm start`

## Обновление контента
После изменения файлов:
- Если меняли только контент (тексты, картинки) - перезапустите сервер
- Если меняли код - пересоберите: `npm run build`
