# ✅ Правильная настройка Vercel проекта

## Шаг за шагом

### 1. Удалите старый проект (если еще не удалили)
- Vercel Dashboard → проект 1mb3-guide-2026
- Settings → General → Delete Project

### 2. Создайте новый проект

1. Откройте https://vercel.com/new
2. Нажмите "Import Git Repository"
3. Найдите `berdniko9544-gif/1mb3-guide-2026`
4. Нажмите "Import"

### 3. ВАЖНО! Настройки перед деплоем

На странице "Configure Project" **НЕ ТРОГАЙТЕ НИЧЕГО!**

Оставьте все поля **ПУСТЫМИ**:
- ✅ Framework Preset: **Next.js** (должно определиться автоматически)
- ✅ Root Directory: **./** (оставьте как есть)
- ✅ Build Command: **пустое** (Vercel сам знает: `next build`)
- ✅ Output Directory: **пустое** (Vercel сам знает: `.next`)
- ✅ Install Command: **пустое** (Vercel сам знает: `npm install`)

**НЕ ВКЛЮЧАЙТЕ "Override"!** Просто оставьте все как есть.

### 4. Деплой

Просто нажмите большую кнопку **"Deploy"**

Vercel автоматически:
- Определит Next.js
- Установит зависимости
- Соберёт проект
- Задеплоит

### 5. Подождите

Сборка займёт 1-2 минуты. После завершения сайт будет доступен по адресу:
`https://1mb3-guide-2026.vercel.app`

---

## Если всё равно ошибка

Если Vercel не может найти Next.js, попробуйте:

1. **Вариант A:** Выберите Framework Preset вручную
   - В настройках выберите "Next.js" из списка
   - Оставьте остальные поля пустыми
   - Deploy

2. **Вариант B:** Укажите Node.js версию
   - В Root Directory проекта создайте файл `.node-version`
   - Содержимое: `20`
   - Commit и push
   - Попробуйте деплой снова

---

## Почему возникла проблема?

Скорее всего при первом деплое были установлены неправильные "Production Overrides" которые переопределили настройки проекта. Поэтому проще всего - удалить проект и создать заново с правильными настройками.
